# Chrome Extension Toolkit

Everything you need to build, ship, and monetize Chrome extensions.

## Starter Templates

| Template | Description |
|----------|-------------|
| [chrome-extension-react-starter](https://github.com/theluckystrike/chrome-extension-react-starter) | React 18, TypeScript, Vite, Tailwind |
| [chrome-extension-svelte-starter](https://github.com/theluckystrike/chrome-extension-svelte-starter) | Svelte 4, TypeScript, Vite, Tailwind |
| [chrome-extension-vue-starter](https://github.com/theluckystrike/chrome-extension-vue-starter) | Vue 3, TypeScript, Vite, Tailwind |
| [chrome-extension-vanilla-ts-starter](https://github.com/theluckystrike/chrome-extension-vanilla-ts-starter) | Pure TypeScript with esbuild |
| [chrome-extension-content-script-starter](https://github.com/theluckystrike/chrome-extension-content-script-starter) | Focused on DOM injection |
| [chrome-extension-popup-starter](https://github.com/theluckystrike/chrome-extension-popup-starter) | Focused on popup UI |
| [chrome-extension-devtools-starter](https://github.com/theluckystrike/chrome-extension-devtools-starter) | Custom Chrome DevTools panels |
| [chrome-extension-side-panel-starter](https://github.com/theluckystrike/chrome-extension-side-panel-starter) | Persistent side panel UI |
| [chrome-extension-full-stack](https://github.com/theluckystrike/chrome-extension-full-stack) | Svelte, Vite, Tailwind, messaging, storage |
| [chrome-extension-mv3-minimal](https://github.com/theluckystrike/chrome-extension-mv3-minimal) | Pure HTML/JS/CSS, no bundler |
