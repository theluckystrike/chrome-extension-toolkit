Starter Templates

Ten distinct, ready-to-use boilerplate repositories. Clone them, npm install, and start building immediately.

React Starter

React 18 + TypeScript + Vite + Tailwind CSS boilerplate. MV3 ready.

Svelte Starter

Svelte 4 + TypeScript + Vite + Tailwind CSS boilerplate. MV3 ready.

Vue Starter

Vue 3 + TypeScript + Vite + Tailwind CSS boilerplate. MV3 ready.

Vanilla TS Starter

Pure TypeScript + esbuild Chrome extension boilerplate. No heavy frameworks.

Content Script Starter

Pure TypeScript boilerplate explicitly focused on injecting DOM logic via Content Scripts.

Popup Starter

Pure TypeScript boilerplate explicitly focused on the popup UI.

DevTools Starter

Pure TypeScript boilerplate designed for creating custom Chrome Developer Tools panels.

Side Panel Starter

Pure TypeScript boilerplate for the powerful new `chrome.sidePanel` persistent UI standard.

Full-Stack Starter

The kitchen sink. Svelte, Vite, Background messaging, injected UIs, and Storage Sync APIs all configured to work together.

Minimal MV3 Starter

The absolute bare minimum Manifest V3 extension. Pure HTML/JS/CSS. No bundler.

Helper Packages

Robust npm packages to abstract away the most painful aspects of extension development.

webext-storage

A strongly-typed, promise-based wrapper around `chrome.storage` that handles serialization and quota limits.

npm install @theluckystrike/webext-storage

webext-messaging

A robust messaging bridge between your background scripts, content scripts, and UI layers.

npm install @theluckystrike/webext-messaging

webext-permissions

Simplified API request patterns for optional host permissions and extension features.

npm install @theluckystrike/webext-permissions
